About this game

This game is based on RPGCat and if you're seeing this content, the About page hasn't been updated yet. What follows is information about the RPGCat framework and not this specific game.

RPGCat

RPGCat is a project to create a browser based RPG along the lines of Shartak and UrbanDead. It's written in Perl and uses the Perl modules Catalyst, Template::Toolkit and DBIx::Class (among many others) to provide a framework which can be modified and updated easily.

Right now there's not a lot to see but if you think you can help us, get in touch.
